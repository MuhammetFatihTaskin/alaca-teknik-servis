# 🔧 Alaca Teknik Servis Yönetim Sistemi

Teknik servis işletmeleri için geliştirilmiş, tek sayfalık web tabanlı yönetim uygulaması.

## 🚀 Özellikler

- Müşteri ve cihaz kayıt sistemi
- Servis takip ve durum yönetimi
- Barkod yazdırma (müşteri & servis kopyası)
- Cari hesap yönetimi
- Çok kullanıcılı giriş sistemi (admin / teknisyen)
- Termal yazıcı desteği

## 🛠 Kullanılan Teknolojiler

- HTML5 / CSS3 / Vanilla JavaScript
- JsBarcode kütüphanesi
- LocalStorage tabanlı veri yönetimi

## 📌 Kullanım

Projeyi klonla veya `WUHU.html` dosyasını doğrudan tarayıcıda aç.
